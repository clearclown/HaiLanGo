# UI/UX設計書 - LinguaAI

## 1. デザインコンセプト

### 1.1 参考アプリ：abceed
**採用する主要な特徴**
- クリーンでミニマルなデザイン
- 1ページ1画面の集中型学習体験
- 大きく押しやすいボタン
- 明確な進捗表示
- 直感的なナビゲーション

### 1.2 ターゲットユーザー
- **学生**（高校生〜大学生）：第二外国語学習、留学準備
- **社会人**：ビジネス・旅行のための語学習得
- **言語学習初心者**：初めて外国語を学ぶ人

### 1.3 デザイン原則
1. **シンプル**：複雑な操作を避け、ワンタップで完結
2. **集中**：学習中は1つの課題に集中できる設計
3. **フィードバック**：アクションに対する即座の反応
4. **達成感**：進捗が視覚的にわかりやすい
5. **アクセシビリティ**：誰でも使いやすいUI

## 3. タイポグラフィ

### 3.1 フォント選定
- **日本語**：Noto Sans JP
- **欧文**：Inter / Roboto
- **アラビア語・ヘブライ語など**：Noto Sans（該当言語）

```

Heading 2: 24px (セクションタイトル)
Heading 3: 20px (カードタイトル)
Body: 16px (本文)
Small: 14px (補足情報)

│                                     │
└─────────────────────────────────────┘
```

### 教師モードの特徴

**自動再生フロー**
1. ページ画像を表示
2. 学習先言語のテキストを読み上げ
3. （設定により）母国語の翻訳を読み上げ
4. （設定により）単語・文法の解説
5. （設定により）発音練習の時間を設ける
6. 設定した間隔（例：5秒）待機
7. 次のページへ自動遷移
8. 1に戻る（全ページ完了まで繰り返し）

**カスタマイズ可能な設定**
- **再生速度**：0.5x, 0.75x, 1.0x, 1.25x, 1.5x, 2.0x
- **ページ間隔**：0秒, 3秒, 5秒, 10秒, 15秒
- **解説の有無**：単語・文法解説を含めるか
- **発音練習の有無**：各フレーズで発音練習を促すか
- **リピート回数**：各フレーズを何回読み上げるか（1-3回）
- **翻訳読み上げ**：母国語訳を読み上げるか

**バックグラウンド再生**
- 画面オフでも再生継続（音声のみ）
- ロック画面からのコントロール可能
- 通知センターでの再生コントロール

**オフライン対応**
- 事前にダウンロードボタン追加
- 教師モードの音声データを全ページ分ダウンロード
- オフラインで完全に再生可能

- **削除時**：左右どちらかにスワイプアウト
- **時間**：0.4秒

- **更新時**：スムーズに伸びる
- **完了時**：緑色に変化 + チェックマークアイコン表示

### 6.5 音声再生
- **再生中**：パルス（波紋）アニメーション
- **録音中**：赤い円が拡大・縮小を繰り返す
