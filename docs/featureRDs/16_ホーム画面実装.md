# 機能実装: ホーム画面

## 要件

### 機能要件

#### ホーム画面機能
1. **ウェルカムメッセージ**
   - ユーザー名表示
   - モチベーション向上のメッセージ

2. **今日の学習カード**
   - 進捗バー表示
   - 「続きから学習」ボタン
   - 完了ページ数表示

3. **クイックアクセス**
   - マイ本へのショートカット
   - 復習へのショートカット

4. **学習統計**
   - 連続学習日数
   - 総学習時間
   - ストリーク表示

### 非機能要件

- **パフォーマンス**:
  - 画面表示: 1秒以内
  - データ取得: 500ms以内

- **セキュリティ**:
  - ユーザー認証必須

## 実装指示

### Step 1: テスト設計

1. **ユニットテスト（Vitest）**
   - コンポーネントのレンダリング
   - データ表示ロジック
   - ナビゲーション

2. **統合テスト（Vitest）**
   - API呼び出し
   - データ取得・表示

3. **E2Eテスト（Playwright）**
   - ホーム画面の表示
   - クイックアクセスの動作
   - 学習開始フロー

テストファイルは `frontend/web/app/(home)/page.test.tsx` および `frontend/web/e2e/home.spec.ts` に配置。

### Step 2: 実装

#### 実装ファイル構造

```
frontend/web/
├── app/
│   └── (home)/
│       ├── page.tsx                 # ホーム画面
│       └── page.test.tsx            # Vitestテスト
├── components/
│   └── home/
│       ├── WelcomeCard.tsx          # ウェルカムカード
│       ├── TodayLearningCard.tsx    # 今日の学習カード
│       ├── QuickAccess.tsx          # クイックアクセス
│       ├── LearningStats.tsx        # 学習統計
│       └── *.test.tsx               # Vitestテスト
└── e2e/
    └── home.spec.ts                 # Playwrightテスト
```

#### APIエンドポイント

```
GET /api/v1/home/dashboard
GET /api/v1/home/stats
```
